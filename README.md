## CRUD fullStack

![](img/fullStack.jpg)
Un CRUD fullStack, realizado con React, node y Mysql, es original del canal [informaticaDP](https://www.youtube.com/c/Inform%C3%A1ticaDP);
con algunas modificaciones personales.
Dentro de la carpeta node se encuentra una carpeta llamada "bk", que contiene la base de datos en mysql, para que la importes si descargas el proyecto (debes tener algun paquete de servidor web como XAMPP รณ WAMP).
